# class-management-system

`班务管理系统`

# V1.0 成绩录入 ✅
## 1.快速录入成绩
- 通过pinyin-pro库进行达到快速姓名匹配
- 回车键自动提交，重新聚焦到姓名输入框
## 2.支持excel导出，方便copy成绩
## 3.支持自定义导入学生名单
## 4.支持刷新，浏览器持久化保存信息

# V2.0 期末评语 ✅

## 2.支持字体大小、落款名等配置
## 3.支持PDF导出
